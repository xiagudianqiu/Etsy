# 用户使用指南

> 给 Etsy 卖家用户的简明使用手册（约 5 分钟读完）

---

## 第一次使用

### 1. 注册账号
- 打开应用网址
- 点「注册」→ 填邮箱 + 密码（至少 6 位）
- 注册后自动登录

### 2. 配置产品成本
- 右上角点齿轮 ⚙ 打开设置
- 在「币种与汇率」选你习惯的币种（如人民币 ¥）
- 点「刷新汇率」获取最新汇率
- 暂时跳过产品成本（等导入后让系统自动发现）

### 3. 导入第一份 CSV
- 右上角点「导入」
- 拖拽你的 Etsy Monthly Statement CSV（文件名格式：`etsy_statement_2026_4.csv`）
- 可一次性拖入多个月份
- 等待几秒看到 ✓ 成功

### 4. 配置产品成本（识别后再做）
- 回到设置 → 产品成本
- 会看到「发现 N 个未配置产品」提示
- 点「全部添加」→ 给每个产品输入成本（人民币）
- 点「保存」

利润数据立即更新。

## 日常使用

### 切换月份
顶部月份栏：
- **点单个月** → 看那个月的数据
- **多选** → 点多个月份合并
- **点「全部 N 月」** → 看所有月份合计

### 切换币种
顶栏右侧有币种切换器：`$ 美元` / `¥ 人民币` / `€ 欧元`，切了所有金额立刻按汇率转换。

### 查看销售日历
仪表盘下方有「销售日历」热力图：
- 金色越亮 = 那天单越多
- 鼠标悬停看具体订单
- 右上角箭头翻月份

### 查看哪个产品最赚钱
仪表盘的「产品利润分析」表：
- 按净利润降序
- 第一名带 🏆 金色徽章
- 每行显示销量、销售额、费用、成本、利润、利润率

### 模拟成本变化
切到「成本与广告」页：
- **成本敏感度模拟器** 可以拖滑块看：
  - 如果产品成本涨 ¥10，利润变多少
  - 如果汇率变化 0.2，利润变多少

### 健康度评分
仪表盘的「运营健康度」（圆环数字）：
- 80+ = 优秀
- 60-80 = 良好
- 40-60 = 注意
- < 40 = 需改善

综合 4 维：利润率 / 广告占比 / 退款率 / 利润规模

## 邮件备份（可选）

让系统每天自动把当天上传的 CSV 备份到你邮箱。

### 开启
1. 设置 → 邮件备份 → 开关打开
2. 填收件邮箱
3. 点「测试入队」验证（测试文件当晚 23:00 发出）

### 工作机制
- 上传 CSV → 暂存到服务器队列（不立即发）
- **每天晚上 23:00**（北京时间）→ 自动把当天所有 CSV 合并成一封邮件发到你邮箱
- 邮件附件包含所有 CSV + 摘要

### 注意
- 每月最多发 31 封（默认配额）
- 关闭开关即停止发送
- 队列里未发送的邮件第二天 23:00 才会发

## 导出报表

### CSV 导出
月份栏点「CSV」→ 下载该月详细 Excel（销售/费用/订单明细）

### PDF 导出
月份栏点「PDF」→ 生成专业报告（含图表/数据/分析）
- 多月模式下导出合并报告
- 实时进度提示
- 浅色专业风，可打印

## 配额说明

侧边栏底部显示本月配额：

| 配额 | 默认 | 用完会怎样 |
|------|------|-----------|
| CSV 上传 | 30 次/月 | 提示「配额已用完」，下月 1 号自动重置 |
| 邮件发送 | 31 封/月 | 同上 |

**配额按用户独立计算**，每月 1 号 00:00 北京时间自动重置。

## 常见问题

### Q: CSV 导入后数据是 0？
**A**: 检查文件名格式。必须是 `etsy_statement_YYYY_M.csv`（年份和月份）。

### Q: 注册收不到验证邮件？
**A**: 部署者可能未开启邮箱验证。如果开启了，检查垃圾邮件箱。

### Q: 产品名识别不对？
**A**: 系统按订单标题自动识别。如果不准，可以在设置里手动改产品名（删了重加）。

### Q: 累计利润和单月对不上？
**A**: 累计 = 所有选中月份的合计。先确认月份选择是否正确。

### Q: 邮件没在 23:00 收到？
**A**: 检查：
1. 邮件开关是否打开
2. 收件邮箱是否填对
3. 当天是否有上传 CSV（没上传不会发空邮件）
4. 等 30 分钟（Cron 有时小延迟）
5. 看垃圾邮件箱

### Q: 我能看到其他用户的数据吗？
**A**: 不能。系统用 Supabase RLS 从数据库层强制隔离，每个用户只能看到自己的数据。

### Q: 怎么导出我的所有数据？
**A**: 选「全部 N 月」→ 点 PDF 或 CSV 导出。或者联系部署管理员从数据库直接拉。

### Q: 怎么删除我的账号？
**A**: 联系部署管理员（你的服务提供者）。Supabase Auth 自带删除账号功能。

## 隐私

- 所有 Etsy 销售数据**只有你能看到**（Supabase Row Level Security 保证）
- 开启邮件备份后，CSV 内容会经过 Resend 邮件服务发送（一次性，不持久存储）
- 关闭邮件备份后，数据只存在 Supabase 数据库
- 不会被分享给任何第三方分析服务
